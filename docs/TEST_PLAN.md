# Plano de Testes — Donas OS 2.0 Core

## Teste obrigatório após abrir esta versão
1. Verificar se estoque aparece.
2. Verificar se clientes aparecem.
3. Verificar se histórico aparece.
4. Criar pedido com prato completo.
5. Criar pedido somente com bebida.
6. Editar pedido.
7. Alterar status até Entregue.
8. Conferir financeiro.
9. Exportar backup.
10. Importar backup.

## Critério de aprovação
A versão só deve ser considerada aprovada se dados anteriores aparecerem normalmente e o fluxo principal de pedido continuar funcionando.
