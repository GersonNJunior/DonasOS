# Checklist GO LIVE — Donas OS / Donas Online

Use este checklist antes de publicar o link definitivo e gerar o QR Code.

## 1. Configurações
- [ ] Nome da empresa está correto.
- [ ] WhatsApp definitivo está cadastrado.
- [ ] Instagram está cadastrado.
- [ ] PIX está cadastrado.
- [ ] Taxa de entrega está correta.
- [ ] Tempo médio de preparo está correto.
- [ ] Logo aparece corretamente.

## 2. Online Ready
- [ ] Loja abre e fecha manualmente.
- [ ] Loja automática respeita os horários configurados.
- [ ] Donas Online bloqueia pedidos quando a loja está fechada.
- [ ] Donas Online permite visualizar cardápio mesmo com loja fechada.

## 3. Donas Online — cliente
- [ ] Cardápio aparece com massas, molhos, proteínas, complementos, bebidas e demais itens ativos.
- [ ] Itens sem estoque aparecem como esgotados ou não permitem seleção.
- [ ] Só é possível selecionar 1 massa por prato.
- [ ] Só é possível selecionar 1 molho por prato.
- [ ] Proteínas respeitam a regra de grátis/pagas.
- [ ] Complementos respeitam a regra de grátis/pagos.
- [ ] Cliente consegue adicionar mais de um prato no mesmo pedido.
- [ ] Bebidas podem ser adicionadas sem obrigar massa.
- [ ] Cliente conhecido pelo telefone tem dados preenchidos automaticamente.
- [ ] Cliente novo é obrigado a informar nome e telefone válido.
- [ ] Delivery exige endereço completo, número, bairro e confirmação do endereço.
- [ ] Retirada não exige endereço.
- [ ] Pedido finalizado aparece na aba Pedidos como Aguardando confirmação.
- [ ] Mensagem do WhatsApp abre com o pedido completo.
- [ ] Pedido de retirada não envia endereço na mensagem do WhatsApp.

## 4. Pedidos internos
- [ ] Montar pedido interno permite apenas 1 massa e 1 molho por prato.
- [ ] Pedido interno com múltiplos pratos funciona.
- [ ] Pedido interno entra em Pedido Feito.
- [ ] Pedido online entra em Aguardando confirmação.
- [ ] Pedido online pode ser confirmado e enviado para preparo.
- [ ] Status funcionam: Aguardando confirmação, Pedido Feito, Preparando, Pedido Pronto, Entregue.
- [ ] Pagamento só entra no financeiro quando o pedido for entregue/concluído.
- [ ] Edição de pedido funciona.
- [ ] Exclusão/cancelamento ajusta financeiro e estoque conforme opção escolhida.

## 5. Cozinha
- [ ] Pedidos aparecem separados por status.
- [ ] Pedido muda de status corretamente.
- [ ] Pedido pronto fica visualmente claro.
- [ ] Pedido entregue some ou vai para histórico conforme fluxo atual.

## 6. Estoque
- [ ] Cadastro de item funciona.
- [ ] Edição de item funciona sem precisar rolar para cima.
- [ ] Unidade em un incrementa 1, 2, 3.
- [ ] Unidade em g/kg/ml/L aceita casas decimais.
- [ ] Estoque baixa ao confirmar/preparar o pedido conforme regra atual.
- [ ] Estoque volta ao cancelar pedido quando não houver devolução/perda.
- [ ] Estoque não volta quando marcado como devolução/perda.

## 7. Compras
- [ ] Compra é registrada no histórico.
- [ ] Item comprado pode entrar no estoque.
- [ ] Lançar como saída no financeiro vem marcado por padrão.
- [ ] Ícone da compra acompanha o item quando possível.
- [ ] Valor por unidade/kg/L fica coerente.

## 8. Produção
- [ ] Receita pode ser criada.
- [ ] Produção calcula insumos para a quantidade escolhida.
- [ ] Alterar quantidade atualiza insumos imediatamente.
- [ ] Produção baixa insumos.
- [ ] Produção adiciona produto final ao estoque.

## 9. Clientes
- [ ] Cadastro manual funciona.
- [ ] Busca por telefone no pedido interno funciona sem precisar digitar DDD completo.
- [ ] Alterar endereço só no pedido não altera o cadastro original.
- [ ] Histórico do cliente aparece no Donas Online quando identificado.
- [ ] Repetir último pedido funciona.

## 10. Financeiro e relatórios
- [ ] Pedido entregue entra no financeiro.
- [ ] Compras lançadas como saída aparecem no financeiro.
- [ ] Fechamento de caixa mostra totais por forma de pagamento.
- [ ] Relatórios abrem corretamente.
- [ ] PDF/relatório exportado fica legível.

## 11. Campanhas
- [ ] Campanha interna aparece no Donas OS quando configurada.
- [ ] Campanha do Portal aparece no Donas Online quando configurada.
- [ ] Campanha inativa não aparece.
- [ ] Campanha fora do período não aparece.

## 12. Backup
- [ ] Exportar backup funciona.
- [ ] Importar backup no mesmo navegador funciona.
- [ ] Importar backup em outro navegador funciona.
- [ ] Dados voltam: pedidos, clientes, estoque, compras, produção, financeiro, campanhas e configurações.

## 13. Dispositivos
- [ ] Testar Donas OS no computador principal.
- [ ] Testar Donas Online no celular Android.
- [ ] Testar Donas Online no iPhone, se possível.
- [ ] Testar link no Chrome.
- [ ] Testar link no navegador do Instagram/WhatsApp.

## 14. Teste final real
- [ ] Abrir loja.
- [ ] Fazer pedido pelo Donas Online como cliente novo.
- [ ] Fazer pedido pelo Donas Online como cliente já cadastrado.
- [ ] Fazer pedido de retirada.
- [ ] Fazer pedido de delivery.
- [ ] Abrir WhatsApp com mensagem pronta.
- [ ] Confirmar pedido no Donas OS.
- [ ] Enviar para cozinha.
- [ ] Marcar como pronto.
- [ ] Marcar como entregue.
- [ ] Conferir financeiro, estoque e relatório.
- [ ] Fazer backup final.
