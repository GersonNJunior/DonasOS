# Release 1.1.8 - Ajustes de nomenclatura e navegação do portal

- Remove o atalho de Bebidas do topo do portal do cliente.
- Deixa preparado o atalho de Pratos Prontos para uso futuro quando houver itens nessa categoria.
- Altera a busca de cliente para permitir clicar no nome encontrado antes de vincular o cadastro.
- Renomeia botões do portal: "Adicionar outro prato", "Concluir pedido" e "Limpar pedido".
- Adiciona confirmação antes de limpar o pedido.
- Após limpar o pedido, retorna para o início do portal.
- Ajusta a rolagem do botão Carrinho no rodapé para mostrar o título do carrinho.
