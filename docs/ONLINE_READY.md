# Donas OS 2.1 - Online Ready

## Objetivo
Preparar o sistema para a futura fase online sem ativar sincronização real ainda.

## Incluído
- IDs universais para pedidos, clientes, itens, compras, receitas e produções.
- Estrutura `sync` no banco local.
- Tela Online Ready para configurar modo local/online, loja aberta/fechada e credenciais futuras.
- Checklist técnico da preparação.
- Compatibilidade com LocalStorage atual.

## Futuro
- Supabase Auth.
- Tabelas online.
- Cardápio público.
- Redirecionamento para WhatsApp.
- PWA instalável.
