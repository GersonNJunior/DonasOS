# Donas OS 1.1.0 - Supabase Pedidos Beta

Primeira integração real com Supabase.

## Incluído
- Conexão com Supabase configurada.
- Pedido feito no Donas Online é gravado nas tabelas `clientes`, `pedidos` e `pedido_itens`.
- Donas OS Admin (`?admin=1`) busca pedidos do Supabase automaticamente.
- Atualização periódica dos pedidos online no painel administrativo.
- Online Ready salva dados básicos da loja no Supabase.

## Importante
Esta é uma versão beta da sincronização. O WhatsApp continua como fallback operacional.
