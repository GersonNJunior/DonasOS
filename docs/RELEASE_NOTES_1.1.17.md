# Donas OS 1.1.17

## Correção / melhoria
- Separa status do pedido do status financeiro.
- Adiciona opção de pagamento recebido no pedido interno e no portal.
- Receita entra no financeiro apenas quando o pedido estiver marcado como pago.
- Permite marcar pagamento pelo painel administrativo sem depender da entrega.
- Mantém pedido entregue sem pagamento como pendência financeira.
