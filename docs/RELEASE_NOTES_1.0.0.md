# Donas OS 1.0.0 — Produção GO LIVE

Data: 2026-06-29
Status: Release oficial candidata à produção

## Objetivo
Primeira versão oficial preparada para publicação online e geração do QR Code da Donas da Massa.

## Inclui
- Donas Online homologado para cliente novo e cliente recorrente.
- Retirada sem endereço na mensagem do WhatsApp.
- Delivery com endereço obrigatório.
- Opção de talher no pedido online.
- Banner configurável como informativo ou apenas imagem.
- Tamanho de banner: pequeno, médio, grande ou tela cheia.
- Pedido online limitado a 1 massa e 1 molho por prato.
- Múltiplos pratos no mesmo pedido.
- Bebidas no pedido.
- Campanhas no Donas Online.
- Integração com WhatsApp.
- Base administrativa local preservada.

## Congelamento
Esta versão deve ser usada como base para publicação. Antes da inauguração, alterar apenas bugs críticos, textos indispensáveis, dados da empresa ou ajustes necessários ao funcionamento online.
